Kimray Valve

Launch New Session

KimRay Valve, built for the New Mexico Junior College, is a live 3-D Immersive Maintenance Demonstration built on top of Virtual World Framework to demonstrate the capabilities of Remote Subject Matter Experts interacting with a student using a shared real time lesson workspace utilizing webRTC to deliver real time video and voice over IP without any browser plugins. This demo requires the Chrome (31+) browser.
